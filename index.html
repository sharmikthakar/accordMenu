About accordMenu

One plugin to accordian them all

accordMenu is a plugin that intelligently detects any nested ordered or unordered lists and renders them as vertical accordians.

At default settings accordMenu is just another accordian, however power users can get total control on the behavior of the plugin by tweaking its many options.

How to Use

1. Add reference link to jQuery

Note: Do not add 2 jQuery instances. Preferably use the CDN link
<script type="text/javascript" src="http://code.jquery.com/jquery-1.8.2.min.js"></script>

2. Add reference link to accordMenu

Note: Ensure that jQuery is linked before using accordMenu
<script type="text/javascript" src="js/jquery.accordMenu.min.js"></script>

3. HTML Template

<ul class="accordThis">
  <li>
    <a href="#">Link1</a>
    <div>Content for Link1</div>
  </li>
  <li>
    <a href="#">Link2</a>
    <ul>
	<li>Nested accordian</li>
    </ul>
  </li>
</ul>

Run accordMenu on the list

Note: Currently accordMenu runs only on a Class and Not an ID.
$(".accordThis ").accordMenu();

API and Options

  • Behaviour
  • keepParentOpentrue / falseWhen navigating a child link, do you want to close any existing open siblings of the parent (false) or keep them open (true)?
  • keepSiblingsOpentrue / falseWhen navigating any link, do you want to close any existing open siblings (false) or keep them open (true)?
  • retainOpentrue / falseDo you want any open siblings to retain their current state (true) or close (false).
  • animation'slide' / 'hide'Should there be any transition when switching content (slide) or they should close instantly (hide).
  • Presentation
  • selLinkcolorWhat color should be applied to the active link
  • parLinkcolorWhat color should be applied to the immediate parent link
  • mainLinkcolorWhat color should be applied to the root link
